About & Setup Guide
⚙ PANDUAN SETUP CLOUDFLARE WORKER
1
LOGIN CLOUDFLARE
Pergi ke dash.cloudflare.com dan log masuk ke akaun awak.
2
BUAT WORKER BARU
Klik Workers & Pages → Create Application → Create Worker. Namakan worker awak (contoh: mfx-scanner).
3
PASTE KOD WORKER
Buka fail mfx-worker.js yang diberikan. Copy semua kod dan paste dalam editor Cloudflare. Klik Save and Deploy.
4
SET API KEY (SELAMAT)
Dalam Worker awak → Settings → Variables and Secrets → tambah:
Variable name: VT_API_KEY
Value: [API key VirusTotal awak]
☑ Encrypt (pilih ini!)
5
COPY URL WORKER
URL worker awak akan kelihatan seperti ini. Copy URL tersebut:
https://mfx-scanner.YOURNAME.workers.dev
6
UPDATE FRONTEND
Dalam fail HTML ini, cari baris ini dan tukar kepada URL worker awak:
const WORKER_URL = 'https://YOUR-WORKER.workers.dev';
7
SHARE KE TEAM
Deploy fail HTML ini ke mana-mana hosting (Cloudflare Pages, GitHub Pages, Netlify — semua percuma). Share URL kepada team awak. ✅
